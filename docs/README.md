## Document Conventions

### Code Examples
- **Rust code** is shown in fenced blocks with syntax highlighting
- **Shell commands** are shown with prompt (`$` for user, `#` for root)
- **Configuration files** are shown with comments explaining options

### Warning and Information Boxes

> **Note**: Important information is highlighted in blue boxes.

> **Warning**: Critical information requiring attention.

> **Tip**: Helpful suggestions and best practices.

### Version Information
- This documentation reflects MultiOS version **v1.0.0**
- Last updated: **2025-11-02**
- Compatible with Rust **1.70+**

## Contributing to Documentation

The documentation is part of the main MultiOS repository. To contribute:

1. Fork the repository
2. Make your changes to documentation files
3. Ensure examples compile and work correctly
4. Submit a pull request

See the [Contributing Guidelines](developer/contributing.md) for detailed instructions.
