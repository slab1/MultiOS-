## ğŸ› ï¸ Development Environment Setup

### Required Tools
```bash
# Install Rust toolchain
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install cross-compilation tools
sudo apt-get install -y gcc-aarch64-linux-gnu gcc-riscv64-linux-gnu

# Install QEMU for testing
sudo apt-get install -y qemu-system-x86 qemu-system-aarch64 qemu-system-riscv64

# Clone MultiOS repository
git clone https://github.com/multios/multiOS.git
cd multiOS

# Build and test
make setup
make test-all
```

- **VS Code**: MultiOS extension pack with Rust and debugging support
