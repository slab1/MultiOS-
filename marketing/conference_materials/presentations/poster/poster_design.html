Abstract

We present MultiOS, an educational operating system designed specifically for teaching operating systems concepts through hands-on, multi-architecture development. MultiOS enables students to learn OS principles by implementing and testing kernel components across x86_64, ARM64, and RISC-V platforms using a single codebase.

Key Innovation: Cross-platform educational OS with real-time visualization and automated assessment

The Challenge

  • Students learn OS concepts in isolation
  • Limited hands-on experience with real kernels
  • Single-platform learning misses modern computing diversity
  • Gap between theory and industry practice
  • Lack of immediate feedback in learning

Research Objectives

  • Design multi-architecture educational OS
  • Create hands-on learning modules
  • Develop real-time visualization tools
  • Enable cross-platform assessment
  • Build instructor support materials

Methodology

  • User-centered design approach
  • Iterative development with student feedback
  • Cross-platform testing framework
  • Educational assessment integration
  • Community-driven content creation
